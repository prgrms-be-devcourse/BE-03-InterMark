=== 스케쥴 리스트 조회

.request
```
GET /api/v1/schedules?musical=&page=&size=
```

====== request parameter

|===
| Parameter |Description

|musical
|조회할 스케쥴들을 가진 뮤지컬의  id

|page
|현재 출력될 페이지 번호

|size
|한 페이지에 출력될 유저
|===

.response
====== response body

```json
{
    "data": [
        {
            "isDeleted": false,
            "musicalName": "테스트 뮤지컬",
            "stadiumName": "예술의전당",
            "startTime": "2023-02-07T00:14:00",
            "endTime": "2023-02-07T01:34:00"
        },
        {
            "isDeleted": false,
            "musicalName": "테스트 뮤지컬",
            "stadiumName": "예술의전당",
            "startTime": "2023-02-07T20:00:00",
            "endTime": "2023-02-07T21:20:00"
        }
    ],
    "nowPageNumbers": [
        1
    ],
    "nowPage": 1,
    "next": false,
    "prev": false
}
```
